# 📊 A/B Testing & Conversion Funnel Analysis

## 🚀 Project Overview

This project analyzes user behavior in an e-commerce platform to identify conversion bottlenecks and evaluate product changes using A/B testing.

It simulates real-world product analytics workflows including funnel analysis, statistical validation, and business decision-making.

---

## 🎯 Objectives

* Analyze user conversion funnel (View → Cart → Purchase)
* Identify drop-off points in the user journey
* Perform A/B testing on user groups
* Validate results using statistical testing
* Provide actionable business recommendations

---

## ⚙️ Tech Stack

* Python (Pandas, NumPy)
* Google Colab
* Matplotlib (visualization)
* Statistics (Hypothesis Testing)

---

## 📊 Funnel Analysis

* View → Cart Conversion: ~4.68%
* Cart → Purchase Conversion: ~60.69%

### 🔍 Insight:

* Significant drop-off (~95%) at the initial stage
* Strong conversion once users reach cart

---

## 🧪 A/B Testing Results

* Conversion A: 6.69%
* Conversion B: 7.18%
* p-value: 0.246

### 📌 Conclusion:

Variant B shows slightly better performance, but the difference is not statistically significant. No strong evidence supports adopting the new version.

---

## 💼 Business Insights

* Majority of users drop at the product interaction stage
* Checkout process is efficient with high conversion
* Key bottleneck lies in view-to-cart transition
* Data inconsistencies observed in event tracking

---

## 🚀 Business Recommendations

* Improve product page UX and engagement
* Add trust signals like reviews and ratings
* Optimize pricing and offers
* Conduct longer and more impactful A/B tests
